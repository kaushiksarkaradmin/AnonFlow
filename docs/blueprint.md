# **App Name**: AnonFlow

## Core Features:

- Anonymous Posting: Allows users to post anonymously to a public timeline.
- IP Address Tracking: Tracks the user's IP address to generate a digital token for unique user identification.
- Digital Token Creation: Generates a unique digital token based on the user's IP address.
- Local Cache Storage: Stores post data and digital tokens in the user's local cache.
- Cache Synchronization: Synchronizes post data and user tokens with other users' caches to maintain a shared timeline.
- Content Moderation Tool: AI moderation tool for filtering inappropriate content, preventing offensive or harmful posts from being displayed in the shared timeline, including flagging or automatic removal.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) for a mysterious and modern feel.
- Background color: Light violet (#EDE7F6), a desaturated version of the primary color for comfortable contrast.
- Accent color: Soft lavender (#D1C4E9) to highlight user interaction elements and guide the user.
- Body and headline font: 'Inter', a sans-serif for clean and accessible reading.
- Simple line icons for navigation and user actions to provide clarity without being intrusive.
- Clean, card-based layout for posts, with clear separation and spacing for easy readability.
- Subtle animations and transitions when loading and displaying new posts to enhance user engagement without being distracting.
